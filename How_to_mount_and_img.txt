How to moung .img with multiple partitions

$ cd /mnt
$ sudo mkdir partition_1
$ sudo mkdir partition_2 #as many partitions as you may need
...
$ fdisk -lu /path/to/img.img
Disk /path/to/img.img: X.X GiB, XXXXXXXXXX bytes, XXXXXXXX sectors
Units: sectors of 1 * 512 = 512 bytes
Sector size (logical/physical): 512 bytes / 512 bytes
I/O size (minimum/optimal): 512 bytes / 512 bytes
Disklabel type: dos
Disk identifier: 0x00000000

Device    Boot  Start      End  Sectors  Size Id Type
/img.img1 *      XXXX   526335   XXXXXX  xxxM  c W95 FAT32 (LBA)
/img.img2      526336 XXXXXXXX XXXXXXXX  x.xG 83 Linux

## Multiple the "Start" of the partition you need by 512
$ sudo mount -o loop,offset=269484032 /path/to/img.img /partition_2

Done!!
